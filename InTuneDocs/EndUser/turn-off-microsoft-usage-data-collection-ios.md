# Inaktivera Microsofts insamling av användningsdata

Microsoft samlar automatiskt in anonymiserade data om företagsportalappens tillförlitlighet och prestanda och hur den används för att kunna förbättra sina produkter och tjänster. 

Du kan inaktivera insamlingen av dessa data på en iOS-enhet med hjälp av inställningen **Användningsdata** i företagsportalappen. IT-administratörer har ingen kontroll över denna datainsamling och kan inte ändra din valda inställning.
